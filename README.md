# Requerimentos
 * Python 2.7 
 * Paramiko Modulo
 * Cryptography Modulo
# Tutorial
 * Atualizar o setuptools do python: sudo pip install --upgrade setuptools
 * Instalar o Paramiko: sudo pip install Paramiko
 * Instalar o Cryptography: sudo pip install cryptography
